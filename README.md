# Тестирование классов Product и Category

## Описание

В проекте реализованы два класса:

- `Product` — описывает товар
- `Category` — описывает категорию товаров

Для них написаны тесты с использованием `pytest`.

---

## Что проверяется

### Класс Product

- корректная инициализация атрибутов:
  - `name`
  - `description`
  - `price`
  - `quantity`
- корректная работа методов:
  - `__str__`
  - `__repr__`

### Класс Category

- корректная инициализация:
  - `name`
  - `description`
  - `products`
- корректный подсчёт:
  - общего количества категорий (`category_count`)
  - общего количества продуктов во всех категориях (`product_count`)
